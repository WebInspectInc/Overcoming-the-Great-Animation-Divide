Don’t be a lazy ginger!

FLIP your animations

First, last, inverse, play.

The world’s smallest example™


.thing {
  width: 500px;
  translate: transform 100ms ease-out;
}
.thing:hover {
  transform: scale(1.2);
}

This is BAD

The world’s smallest good example


.thing {
  width: 600px;
  transform: scale(0.83);
  translate: transform 100ms ease-out;
}
.thing:hover {
  transform: scale(1);
}
